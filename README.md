# Pernille Performance

Clean web project for Pernille Performance.

## Local Development

```bash
npm install
npm run dev
```

## Production

The production site is deployed through Vercel from the `main` branch.

Primary domain:

```text
https://pernilleperformance.com
```

Canonical `www` route:

```text
https://www.pernilleperformance.com
```
